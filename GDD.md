# 🌶️ 胡椒家防衛戦 — Game Design Document v1.0

> **Genre:** Vampire Survivors-like / Auto-battler Roguelite
> **Platform:** Web (HTML5) — 靜態部署於娜娜工坊
> **Art Style:** 16-bit Pixel Art
> **Engine:** Phaser 3 + bitECS
> **Session Length:** 15 分鐘 / 局
> **Status:** Pre-production
> **Created:** 2026-03-30

---

## 1. 核心概述

### 1.1 Elevator Pitch
「胡椒少佐的家被怪物圍攻了！來胡椒家玩的朋友們拿起武器，用各自的絕活守護這個歡樂的聚會所！」

### 1.2 世界觀
胡椒少々塩少佐的家是 VTuber 界知名的聚會聖地，大家都愛來這裡玩老遊戲、聊天、開趴。某天夜裡，不明怪物開始從四面八方湧入——朋友們決定用自己的方式守護胡椒家。

### 1.3 核心循環
```
選角色 → 自動攻擊（+1-2主動技能）→ 擊殺敵人掉經驗寶石
→ 升級選武器/道具 → 變強 → 更多敵人 → 進化合成 → 15分鐘存活
```

### 1.4 設計原則
- **趣味 > 平衡** — 每個流派都要爽，寧可 OP 不可無聊
- **角色即記憶點** — 玩起來感受截然不同，不是換皮
- **驚喜感** — 進化合成有隱藏路線，發現的瞬間要興奮
- **輕量可及** — 網頁打開就玩，不需安裝

## 2. 角色設計

### 2.1 設計哲學：五感差異化
每個角色的「攻擊節奏」完全不同，不只是數值差異。

### 2.2 角色一覽

#### 🎯 ムーさん — 佈陣型
- **概念：** Overwatch 射擊風格，放置砲台填滿畫面
- **主武器：** 吉祥物砲台 — 放置自動瞄準砲塔，同時存在最多 3 座（可升級）
- **手感：** 戰略性佈陣，看砲台交叉火力覆蓋的滿足感
- **主動技能：** 吉祥物變形 — 所有砲台齊射 3 秒，冷卻 25 秒
- **被動：** 砲台存在越久傷害越高（+5%/5秒，上限+30%）
- **像素特徵：** 多種外貌的吉祥物跟隨（2-3像素小夥伴）

| 等級 | 砲台數 | 砲台射程 | 砲台傷害 |
|------|--------|----------|----------|
| 1 | 1 | 80px | 8 |
| 3 | 2 | 90px | 12 |
| 5 | 3 | 100px | 16 |
| 7 | 3 | 120px | 22 |

#### 🐱 にゃおぬこ — 壓迫型
- **概念：** Apex 式貼臉輸出，高頻近距離連擊
- **主武器：** 連射爪擊 — 前方扇形快速揮擊，攻速極快
- **手感：** 腎上腺素飆升的貼臉壓迫，敵人越密越爽
- **主動技能：** Apex 瞬移突進 — 短距離衝刺＋到達時 AOE 爆發，冷卻 15 秒
- **被動：** 連續命中提升攻速（每 10 hit +3%，上限+30%，2秒不命中歸零）
- **像素特徵：** 貓耳 + 尾巴動態擺動

| 等級 | 扇形角度 | 攻速(ms) | 單次傷害 |
|------|----------|----------|----------|
| 1 | 60° | 300 | 5 |
| 3 | 75° | 250 | 7 |
| 5 | 90° | 200 | 9 |
| 7 | 120° | 150 | 12 |

#### 🐕 あばれんぼー — 節奏型
- **概念：** 沒耳朵的柴犬，超長尾巴是武器
- **主武器：** 尾巴迴旋 — 360° 掃擊，蓄力越久範圍越大
- **手感：** 蓄力→釋放的節奏感，大範圍清場的爽快
- **主動技能：** 尾巴無限伸長 — 5秒內尾巴範圍×3，冷卻 30 秒
- **被動：** 蓄力期間減傷 20%（鼓勵站穩蓄力）
- **像素特徵：** 沒耳朵 + 超長尾巴（占角色 sprite 3 倍長）

| 等級 | 基礎範圍 | 最大蓄力範圍 | 蓄力時間 | 傷害 |
|------|----------|-------------|----------|------|
| 1 | 40px | 80px | 1.5s | 15 |
| 3 | 50px | 110px | 1.3s | 22 |
| 5 | 60px | 140px | 1.1s | 30 |
| 7 | 70px | 180px | 0.9s | 40 |

#### 🏍️ 赤穂みのり — 衝刺型（Phase 2）
- **概念：** 速度狂人，移動本身就是攻擊
- **主武器：** 重機軌跡 — 移動時留下傷害帶，速度越快傷害越高
- **手感：** 停不下來的爽快，畫面到處是輪胎痕
- **主動技能：** 全速衝鋒 — 3秒無敵+速度×2+軌跡寬度×3，冷卻 20 秒
- **被動：** 移動速度 +15%（基礎就比其他人快）
- **像素特徵：** 安全帽 + 移動時煙塵特效

#### 🌶️ 胡椒少々塩少佐 — 召喚型（Phase 2 隱藏角色）
- **概念：** 人脈王、老遊戲玩家，召喚各種復古遊戲角色
- **主武器：** 懷舊召喚 — 隨機丟出復古遊戲角色當投射物（每次外觀/軌跡不同）
- **手感：** 混沌中的驚喜，「這次會丟出什麼！」
- **主動技能：** 人脈召喚 — 召喚一位隨機友情援軍（NPC 版其他角色），持續 10 秒
- **被動：** 家主加護 — 在胡椒家地圖上全屬性 +10%
- **解鎖條件：** 用其他 4 位角色各通關一次
- **像素特徵：** 待確認（需要更多胡椒少佐的形象參考）

## 3. 武器系統

### 3.1 架構
全 data-driven，武器定義為 JSON descriptor：
```json
{
  "id": "moo_turret",
  "name": "吉祥物砲台",
  "owner": "ムーさん",
  "type": "turret",
  "damage": 8,
  "cooldown": 800,
  "pattern": "auto_aim",
  "maxLevel": 8,
  "evolves": [
    { "requires": "homing_scope", "into": "auto_sentry_v2" }
  ]
}
```

### 3.2 可獲得武器（升級時隨機出現）

| 武器 | 類型 | 原屬角色 | 說明 |
|------|------|----------|------|
| 吉祥物砲台 | turret | ムーさん | 放置自動砲台 |
| 連射爪擊 | melee | にゃおぬこ | 前方扇形連擊 |
| 尾巴迴旋 | area | あばれんぼー | 蓄力360°掃擊 |
| 重機軌跡 | trail | みのり | 移動留傷害帶 |
| 懷舊召喚 | projectile | 胡椒少佐 | 隨機投射物 |
| 彈幕花火 | projectile | 通用 | 放射狀子彈 |
| 護盾衝擊 | area | 通用 | 身周持續傷害圈 |
| 雷射光束 | beam | 通用 | 直線貫穿 |

每把武器最高 Lv8，升級提升數值 + 視覺效果。

## 4. 道具系統

### 4.1 設計原則：道具是「動詞」
同一道具對不同角色產生不同質變。

### 4.2 道具一覽（MVP 8 種）

| 道具 | 基礎效果 | 對不同角色的質變 |
|------|----------|-----------------|
| 加速帶 | 移動速度+15% | みのり：軌跡寬×2 / ムーさん：砲台變移動砲台 |
| 追蹤晶片 | 投射物追蹤敵人 | にゃおぬこ：爪擊追加追蹤飛爪 / あばれんぼー：尾巴末端發射追蹤彈 |
| 增幅水晶 | 傷害+20% | 胡椒：召喚物體積×2 / ムーさん：砲台多一管砲 |
| 分裂核心 | 投射物擊中分裂 | あばれんぼー：尾巴掃擊產生衝擊波 / にゃおぬこ：爪擊追加分裂爪痕 |
| 冰霜之心 | 攻擊附帶減速 | みのり：軌跡變冰道（敵人滑行） / 胡椒：召喚物爆炸為冰圈 |
| 吸血面具 | 擊殺回血 2% | 通用效果 |
| 經驗磁鐵 | 拾取範圍+30% | 通用效果 |
| 幸運骰子 | 暴擊率+8% | 通用效果 |

道具最高 Lv5，疊加效果遞減（flat_add → percent_mul → cap）。

## 5. 進化合成系統

### 5.1 三層設計

#### 🔓 明牌進化（MVP）
武器 Lv8 + 特定道具 → 進化武器。圖鑑有暗示。

| 武器 | + 道具 | = 進化武器 | 效果 |
|------|--------|-----------|------|
| 吉祥物砲台 | 追蹤晶片 | 全自動哨兵 V2 | 砲台追蹤+連射+範圍擴大 |
| 連射爪擊 | 加速帶 | 影分身亂舞 | 產生殘影，多重判定 |
| 尾巴迴旋 | 增幅水晶 | 暴風尾擊 | 蓄力取消，變持續高速旋轉 |
| 彈幕花火 | 分裂核心 | 超新星爆發 | 子彈爆炸＋碎片二次傷害 |
| 護盾衝擊 | 冰霜之心 | 絕對零度領域 | 持續凍結周圍+傷害提升 |

#### 🔒 隱藏進化（Phase 2）
跨角色武器 + 特定道具，接近觸發時兩個 icon 微微發光。

#### ⭐ 友情進化（Phase 2）
集齊全角色武器碎片 → 專屬大絕演出。

## 6. 敵人設計

### 6.1 設計原則
- 敵人外觀：胡椒家的「不速之客」（待定主題——幽靈？bug？病毒？）
- 基礎小怪只有「追蹤玩家」行為，靠數量形成壓力
- 精英怪和 Boss 有特殊行為

### 6.2 敵人類型

| 類型 | 行為 | 血量 | 速度 | 出現時機 |
|------|------|------|------|----------|
| 雜魚 A | 直線追蹤 | 低 | 慢 | 0:00 起 |
| 雜魚 B | 直線追蹤 | 中 | 中 | 2:00 起 |
| 快速型 | 高速追蹤 | 極低 | 快 | 3:00 起 |
| 坦克型 | 緩慢追蹤 | 高 | 極慢 | 5:00 起 |
| 射手型 | 保持距離+投射物 | 中 | 中 | 7:00 起 |
| 精英怪 | 特殊技能+光環 | 極高 | 中 | 5分鐘 Boss |
| 最終 Boss | 多階段+機制 | 超高 | 變化 | 14:30 |

### 6.3 波次結構
```
0:00-2:00  — 暖身期（雜魚 A，少量）
2:00-5:00  — 升溫期（混合雜魚，數量增加）
5:00       — ⚡ 第一次精英怪（中期爆點 #1）
5:00-10:00 — 壓力期（全種類，密度提升）
10:00      — ⚡ 第二次精英怪（中期爆點 #2）
10:00-14:30— 瘋狂期（大量+高速，考驗 build）
14:30-15:00— 🔥 最終 Boss（存活=通關）
```

## 9. 技術規格

### 9.1 架構
- **引擎：** Phaser 3.80+
- **ECS：** bitECS（實體/組件管理）
- **渲染：** WebGL（Canvas fallback）
- **音效：** Howler.js（object pooling）
- **碰撞：** Spatial Hash Grid
- **物件池：** 全部運行時實體（子彈、敵人、經驗寶石、特效）

### 9.2 性能目標
| 指標 | 目標 |
|------|------|
| 同屏敵人 | 200-400（MVP），上限 800 |
| FPS | 60fps（桌面），30fps（手機） |
| 載入時間 | < 3 秒 |
| 總資源大小 | < 5MB |

### 9.3 數據格式
- 武器定義：`data/weapons.json`
- 道具定義：`data/items.json`
- 進化配方：`data/recipes.json`
- 敵人定義：`data/enemies.json`
- 波次定義：`data/waves.json`

所有遊戲內容由 JSON 驅動，修改不需改代碼。

### 9.4 部署
- 靜態 HTML/JS/CSS
- 娜娜工坊 `Playground/pepper-house-survivors/`
- 入口：`index.html`

## 10. 素材規格

### 10.1 Sprite 規格
- **格式：** PNG，texture atlas（Aseprite → TexturePacker）
- **角色：** 32×32px，4方向 × 4幀走路 + 攻擊動畫
- **敵人：** 16×16px（雜魚）/ 32×32px（精英）/ 64×64px（Boss）
- **武器特效：** 依類型，16-64px
- **UI：** 獨立 sheet

### 10.2 素材清單（MVP）
| 類別 | 數量 | 規格 |
|------|------|------|
| 角色 sprite（3位）| 3 × ~20幀 | 32×32 |
| 敵人 sprite（6類）| 6 × ~8幀 | 16-64px |
| 武器特效 | ~15 組 | 16-64px |
| 道具 icon | 8 | 16×16 |
| 地圖 tileset | 1 組 | 16×16 tiles |
| UI 元素 | 1 組 | 各尺寸 |
| 結算卡背景 | 1 | 320×480 |

### 10.3 音效
- BGM：1 首循環曲（chiptune 風）
- 攻擊音效：每武器 1-2 種
- 升級/進化音效
- Boss 警告音
